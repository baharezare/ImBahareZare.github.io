Project 1: Viral Infection Detection with CT-Scan Images

Overview

This project uses convolutional neural networks (CNNs) to detect viral infections in CT-Scan images. By analyzing medical image data, the model aims to improve the speed and accuracy of infection detection.

Objectives

  • Enhance diagnostic capabilities with machine learning
  • Improve detection accuracy for viral infections in CT scans
  • Develop a scalable deep learning model suitable for clinical use

Methodology

Using a COVID-CT-MD dataset, we trained a CNN model on CT scan slices that were converted from DICOM to JPG format. The images were pre-processed and labeled to identify signs of infection, and the final model is built using advanced architectures such as VGG and other convolutional networks.

Results

The model achieved a high level of accuracy in detecting infections, which has potential applications in clinical settings. Further optimization is planned to improve real-time application in hospital environments.
