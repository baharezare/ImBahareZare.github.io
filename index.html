    Web Development Projects

    Project 4: React Contact List Application

    This project demonstrates a dynamic contact list built with React. It showcases key React concepts such as reusable components, props, array mapping, and dynamic rendering. Styled with modern CSS, the application offers a visually appealing layout to display contact information.

    • Dynamic Data Rendering: The app utilizes an array of objects to simulate a database. Each object contains a contact's name, image URL, phone number, and email.
    • Reusable Components: The ContactCard component is designed to accept props dynamically and display contact details, ensuring modularity and scalability.
    • Responsive Layout: CSS styling ensures the contact list adapts to different screen sizes while maintaining a clean and modern design.
    • Interactive UI: Each contact card displays an image, name, phone, and email, with icons enhancing readability.
