Project 2: Sonic Log Prediction for Drilling Optimization

Overview

This project focuses on predicting sonic log values in drilling data to enhance subsurface understanding. By leveraging machine learning algorithms, the model aims to assist geoscientists in optimizing drilling strategies through accurate log predictions. Also the paper "LEVERAGING DRILLING DATA FOR SONIC LOG PREDICTIONS USING MACHINE LEARNING ALGORITHMS" is submitted on Journal of Petroleum and Science Engineering and is under review.

Objectives

  • Enhance prediction accuracy for sonic log data
  • Optimize drilling decisions with data-driven insights
  • Develop a scalable predictive model applicable to various geological formations

Methodology

Using a large dataset of drilling logs provided by industry partners, we preprocessed and analyzed the data to create predictive models. Machine learning algorithms such as XGBoost, Random Forest, and LSTM were implemented to capture complex patterns in the data and improve sonic log prediction accuracy.

Results

The model achieved a significant improvement in prediction accuracy, enhancing the reliability of subsurface models. This approach has potential applications in real-time drilling optimization, contributing to reduced drilling costs and improved safety.
