Project: Wind Turbine Geospatial Visualization

Overview

This project leverages the Bokeh library to create an interactive geospatial visualization of wind turbine locations. By integrating tile-based maps and converting geospatial data, the visualization provides insights into the geographical distribution of wind turbines.

Tools and Libraries

The visualization is built with the following tools:

  • Bokeh: For interactive plotting and visualization.
  • Pandas: To handle and process the dataset.
  • Pyproj: For transforming geospatial coordinates.
  • Jupyter Notebook: For developing and displaying the analysis interactively.

Dataset Description

The dataset contains geospatial coordinates of wind turbines:

  • Site.Longitude: Longitude of the wind turbine site.
  • Site.Latitude: Latitude of the wind turbine site.
  • Derived x and y coordinates: Coordinates converted to Web Mercator format for mapping.

Visualization

The interactive map displays the wind turbine locations with red markers. Users can zoom and pan to explore the map for detailed spatial analysis.

Key Features

  • Tile provider: CartoDB Positron for an elegant basemap.
  • Interactive tools: Pan, zoom, and reset functionality.
  • Accurate geospatial transformation using EPSG:3857 projection.

Methodology

The following steps were performed to create the visualization:

  1. Imported necessary libraries: Bokeh, Pandas, and Pyproj.
  2. Prepared the dataset by transforming latitude and longitude into Web Mercator coordinates using pyproj.Transformer.
  3. Set up a Bokeh figure with x_axis_type and y_axis_type set to "mercator" for geographical mapping.
  4. Added a tile provider (CartoDB Positron) for the base map layer.
  5. Plotted wind turbine locations using circle markers and adjusted visual properties for clarity.
  6. Rendered the plot using show().

Insights

The geospatial visualization provides actionable insights into the locations of wind turbines:

  • Enables analysis of turbine distribution across geographical regions.
  • Supports identifying high-density clusters of wind turbines.
  • Facilitates decision-making for future site placements based on existing patterns.
